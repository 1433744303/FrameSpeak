# 测试连接功能验证

## 如何测试

1. 访问 http://localhost:5173/
2. 点击右上角"LLM 配置"
3. 填写任意配置（不需要真实的服务）
4. 点击"测试连接"按钮

## 预期行为

### 立即显示（< 1秒）
- 出现旋转的加载图标
- 文字："正在测试连接..."

### 测试完成后（10-15秒内）
成功时：
```
✅ 连接成功！服务可正常使用
```
（绿色背景，5秒后自动消失）

失败时：
```
❌ 连接失败: [具体错误原因]
```
（红色背景，8秒后自动消失）

## 常见错误提示

### Ollama相关
- "无法连接到服务器，请确保服务正在运行"
- "连接超时，请检查网络或服务器地址"

### OpenAI相关
- "API密钥无效，请检查配置"
- "API端点不存在，请检查URL"
- "请求频率超限或配额用完"

### 网络相关
- "网络错误，请检查网络连接"
- "无法解析域名，请检查URL是否正确"

## 调试步骤

如果没有任何提示：

1. 打开浏览器控制台（F12）
2. 切换到 Console 标签
3. 点击"测试连接"
4. 查看是否有错误输出
5. 查看 Network 标签是否有请求发送
